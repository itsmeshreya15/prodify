#  Prodify AI – Task Manager App

 **Live Demo:** https://prodifyy-swart.vercel.app/

---

##  About the Project

**Prodify AI** is a modern full-stack task management application built to help users stay productive and organized.
It allows users to create, update, delete, and manage tasks with a clean and responsive UI.

---

##  Features

*  User Authentication (Login)
*  Add new tasks
*  Edit tasks
*  Delete tasks
*  Mark tasks as completed
*  Dark Mode support
*  Toast notifications (react-hot-toast)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Framer Motion
* React Router DOM
* React Hot Toast

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

### Deployment

* Frontend: Vercel
* Backend: Render

---

##  Project Structure

```
prodify-ai/
├── client/       # Frontend (React + Tailwind)
└── server/       # Backend (Node + Express + MongoDB)
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/itsmeshreya15/prodify-ai.git
cd prodify-ai
```

---

### 2️⃣ Setup Backend

```bash
cd server
npm install
npm start
```

Create a `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
```

---

### 3️⃣ Setup Frontend

```bash
cd client
npm install
npm run dev
```
